# alfafond.com — статичный сайт (3 поверхности, одна папка)

Single-file статика без сборки (шрифты — Google Fonts CDN). Деплой = перетащить
папку `site/alfafond/` целиком.

## Структура (решение 2026-06-10: первое впечатление — EN-нейтральное)

| путь | язык | назначение |
|---|---|---|
| `/` (index.html) | EN | нейтральная обложка-развилка: Research → /ru, Engineering → /dev. Домен НЕ ассоциируется с РФ при первом заходе |

ungrounded AI-контентом — ровно та галлюцинация, против которой позиционируется продукт.
Анти-ассоциация с РФ решена структурой сайта (EN-корень), не фейковым контентом.

удалить (noindex + закрыт в robots.txt).

## Перед публикацией заполнить

1. **`WEB3FORMS_KEY`** в формах `/ru/` и `/dev/` — контакт без раскрытия username/номера автора
   (приватность, выбор пользователя 2026-06-11). Получить бесплатный access key на
   web3forms.com (нужен только e-mail назначения) → вставить вместо `WEB3FORMS_KEY`.
   Форма деградирует без JS (обычный POST). Альтернатива: свой relay/Telegram-бот на
   существующей infra (`services/telegram_alerts.py`) — если объём оправдает.
2. Числа demo-таблицы SBER в `/ru/` (данные на 2026-05-31) — обновить из свежего разбора.
3. GitHub-кнопка снята (приватный код не публикуем); добавить позже только если заведёшь
   публичный профиль с обезличенными демо-репо.
